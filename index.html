Release Highlights

What's New in Version 3.2.1 ✨

⚡

Shizuku 1-Tap Setup & Permission Grant

Automatically grant Restricted Settings (ACCESS_RESTRICTED_SETTINGS), Usage Access (GET_USAGE_STATS), System Overlay, Accessibility, Notification Access, and Battery Whitelist with a single tap using Shizuku.

🛠️

System Service Recovery (`SystemServiceRecovery`)

Automatic background recovery engine for reconnecting system-managed services (Accessibility and Notification Listener) to maintain high uptime and seamless state restoration.

🧹

Clear Test Notifications Button

Dedicated 1-tap "Clear Test Notifications" action added right after the Quick Testing controls to clear test cards and reset active island states instantly.

📱

Direct Notification Listener Toggle Page

Clicking Notification Listener permission setup launches directly into Smart Island's detail toggle page via ACTION_NOTIFICATION_LISTENER_DETAIL_SETTINGS.

🛡️

OEM ROM Background Autostart Guards

Prevents aggressive OEM Accessibility Service kills on MIUI/HyperOS, ColorOS/RealmeUI, FuntouchOS, and OneUI vendor systems.

🎨

Refreshed Midpoint Launcher Icon

Newly styled and balanced app icon with clean proportions and optimized vector assets across all screen densities.

What it does

Notifications that stay useful without taking over your screen.

Floating overlay

Draws a compact, custom overlay above other apps using native Android window manager APIs.

Shizuku 1-Tap Setup

Grant Accessibility, Notification Access, System Overlay, Usage Access, and Battery Whitelist in one tap.

System Service Recovery

Automatic background recovery engine to maintain continuous system service connections.

Notification Suppression

Converts active notifications into glanceable cards while suppressing system shade duplicate alerts.

Landscape Auto-Hide

Automatically detects device orientation to temporarily hide overlay during full-screen games and video streaming.

Lock Screen Privacy

Opt-in lock screen status viewing with automatic sensitive content masking options.

Wavy music seek bar

Renders playing progress as an organic, animated wave with circular thumb drag-to-seek support.

Interactive customization

Adjust island width, height, vertical offsets, corner radius, and custom RGB colors dynamically.

App Shortcuts Launcher

Configure up to eight favorite apps for instant 1-tap launch directly from the expanded island.

Battery Charging & Optimization

Displays animated charging status, time-until-full estimates, and includes background optimization setup.

Local first

Built around transparent permissions.

Smart Island processes notification data on the device, stores preferences locally, and currently does not request the Android INTERNET permission.

SYSTEM_ALERT_WINDOW

Displays the floating island above other apps.

BIND_NOTIFICATION_LISTENER_SERVICE

Reads notification metadata for notification, call, and media states.

FOREGROUND_SERVICE

Keeps the overlay active while Smart Island is enabled.

ACCESS_RESTRICTED_SETTINGS

Allows Shizuku 1-tap permission management on Android 13+ devices.

Getting started

Install, grant, enable.

  1. 01

    Download the APK

    Install the latest signed v3.2.1 package from GitHub Releases.

  2. 02

    Grant overlay permission

    Allow Smart Island to draw the island above other apps (or use 1-Tap Shizuku setup).

  3. 03

    Enable notification access

    Turn on listener access so notifications can become island cards.

  4. 04

    Disable battery optimization

    Set Smart Island to 'No restrictions' so Android keeps the overlay service running.
